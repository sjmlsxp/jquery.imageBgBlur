jquery.imageBgBlur
==================

## 参数说明及默认值

``` js
$(selector).imageBgBlur({
	selector:'img', //图片选择器
	bgColor:"fff",  //文字背景颜色
	opacity:.55,    //文字背景颜色透明度
	strength:6      //图片模糊强度
})
```

## 单个参数调整

``` html
<div class="txt" data-bgcolor="633" data-opacity="0.2" data-strength="0.2"></div>
```
可以通过以上3个data参数，进行单个的调整
